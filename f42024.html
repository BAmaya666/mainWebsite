Using AI tools

I decided to check for AI tools like ChatGPT to help me with productivity and to keep myself updated.

I found some useful websites like Perplexity, Canva, Dalle or Genie for my current projects,but I bet I can find more interesting attributes in more AI tools during the process. Here are the ones I found (all can be used for free):

  • Interactive scenes from Lumalabs: for 3D walkthrough interactions. It’s pretty cool, but it’s only available for iOS systems
  • Google earth studio: Aerial walkthroughs that, if you mix them with the Interactive scenes, you can make pretty cool videos.
  • Perplexity: It’s like ChatGPT, but it lets you upload files, and you can ask questions about it.
  • Scispace: Like ChatGPT but for scientific papers mainly, and you can also upload your files and ask it about them.
  • SGE from Google: Looks amazing, is the Google search bar, but on steroids, it gives you more personalized results. (Update from march 2025: I think SGE became Gemini. Jesus!), but I found the Googlelabs and they have pretty cool apps(?) like this tool to play any music instrument.
  • Leiapix: To add depth to an image and play with it.
  • sketch.metademolab: It animates children’s drawings.
  • Pikalab: Make videos by writing a prompt.
  • Replicate: It creates and restores old pictures, as well as the blurry ones.
  • IllusionDiffusion: Op-art from a picture and a prompt.
  • Ideogram: Generate images from text (it seems more for stickers and prints).
  • DALL-E: Generate images from text.
  • Playground AI: Generate images from text, plus allows you to edit them, all on an artwork. Very cool.
  • Canva: Websites, slides and documents. All of that you can create with the boost of AI.

Figma

I found that if I want to join the UX/UI field, I have to know Figma. Figma is a cloud-based design tool used to create user interfaces, websites, mobile applications, etcetera (it’s like Illustrator and Google Slides combined). As far as I know, Figma is just the visual part of the project, like the storyboard of the user experience. With the addition of real-time collaboration, making it ideal for remote teams. Also, it has a community where you can see other people’s projects, download icons, templates, and such.

I’m watching a course to learn how to use Figma properly, and it seems easy to use, but what I like the most about it is the expertise of the designers. They mentioned the human interface guidelines of Apple and Android for good design to take into consideration for the design of a product.

Also, that it is important to gather a community to touch base, get feedback, and get inspiration from fellow designers.

They also mention that jobs are looking for well-documented projects in a designer’s portfolio. Clear documentation not only demonstrates your skills but also showcases your professionalism and attention to detail. It can be a valuable asset when seeking new clients or job opportunities.

Equally, a great way to get the perfect job for you is to look at what that job is actually looking for and dwell on those aspects (like Figma, Python, and so on).

  • The design of everyday things from Don Norman, a classic that is always good to re-read.
  • Don’t make me think from Steve Krug

Python3

What are Python *args? – Tamara Shatar 2021

In 2021, I was feeling unsure about the path I was taking in the product design industry in Mexico (and because the current populist government is alarmingly ridiculous), so I started considering learning a new skill that I can work anywhere in the world (remotely), that is well paid, and that will be in demand in the future (although the future in the world is foggy with AI). I took programming into consideration; I was amazed by domotics and Arduino back in university. I’m pretty sure I actively looked for a programming course, but there’s a chance the algorithm on Instagram or Youtube offered it to me.

Anyway, I started to take an online HTML and Javascript course, but because I didn’t set a routine to study (at the beginning, I was studying daily, but as time passed, I did it less often), and because I found it challenging (I faced some problems that I couldn’t figure out myself, and coding was a total alien topic to me), it made it harder to follow, plus I wasn’t taking any notes. It got to a point where it was very hard, and I decided to start again a year later, taking notes this time!

My partner (who is a software developer) pointed out that I should learn coding in English instead of Spanish (the course is in Spanish), because that’s how the commands are written originally, and if I wanted to work for a company (in the global north), everything would be in English. Also, he mentioned that Python is the easiest programming language to learn since it is a language similar to English (structurally speaking), so I gave it a shot.

I found The Modern Python 3 Bootcamp course by Colt Steele on Udemy, and it is easy to follow. Also, with my brief experience with HTML and Javascript, I felt more comfortable coding on it. I started using a virtual machine with Linux (my partner told me that’s where most of the software developers program). At the beginning, he installed it on my computer, but I wanted to get my hands dirty and do everything else by myself (Oh my!), so I started to play with this new operating system, and I was trying to give more RAM to the virtual machine, but I messed up and I needed to reinstall it. The screen of the VM looked small, so I had to play with the virtual guest CD image, and it was frustrating, but I figured it out. Also, apparently Linux does everything with commands in its terminal, so I’m learning a lot from the whole situation.

7 segment LED "Maquina del Tiempo" project

"Maquina del Tiempo" repository on GitHub

My partner saw my clear interest in learning Python, and he suggested that it would be handy if I applied my current knowledge to an actual project, so he gave me Maquina del Tiempo, which is to create a LED digital clock using Arduino Uno. So I did my research, and Arduino just works with C++. I have no idea how to work with that language (yet), but Arduino has preset code for certain situations, and I played with it and it worked! But I wanted to program it in Python. There’s some ways to combine them, and guess what? My partner already had a microcontroller for Python, so I’m going to work with it. And funny thing, by the time I’m writing this, I’m learning how to use Python packages to do stuff like code on Arduino, access the internet, and much more.

Now that I have the useful notes that I wrote, I’m a bit ahead of the other half of the course because I’ve been more consistent in my classes, plus after talking with my partner and friends and searching on Youtube and Reddit, I have a more broad idea of what I can do with this knowledge, like using it to apply for a job as a quality assurance assistance analyst in the short term, automating my house and applications, mixing it with SQL (which is in my plans, though I still need to do more research), or even making videogames in Godot or Sandbox. I’m so excited!
